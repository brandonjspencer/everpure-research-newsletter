Everpure Research API

This site serves normalized research outputs and API endpoints generated from the Everpure Notion pipeline.

Useful endpoints

  • /api/health
  • /api/metadata
  • /api/weeks
  • /api/findings
  • /api/summary
  • /api/decks
  • /api/deck-summary
  • /api/deck-details
  • /api/deck-content

To rebuild the site automatically, configure a Netlify build hook and set NETLIFY_BUILD_HOOK_URL plus REFRESH_TRIGGER_SECRET.
